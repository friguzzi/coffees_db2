# jdbc-tutorial

Project for the JDBC tutorial in the course [Technology for Databases](https://unife.coursecatalogue.cineca.it/corsi/2025/320728/insegnamenti/2025/49593/2025/2?annoOrdinamento=2025&coorte=2025) at the University of Ferrara.

Subset of the code from the [Oracle JDBC Basics tutorial](http://docs.oracle.com/javase/tutorial/jdbc/basics/index.html) where the connection is to a [DB2](https://www.ibm.com/products/db2) database
